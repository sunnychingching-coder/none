設定一個開始的按鈕 (注意按鈕不要放在遊戲主圖內)，當按下開始就執行以下邏輯 :
設定 背包 = [ ]    # 這邊也要視覺化背包，注意不要放在遊戲主圖內
進行 for loop 整個 row
    進行 for loop 整個 column
        礦工從左往右，由上而下移動    # 每一格0.3秒
        if 黃金 : 開採礦石，背包 append(黃金)
        else if 水晶 : 背包 append(水晶)
        else if 怪物 : 砍他 !    # 這邊補上砍他的動畫
        if 背包內黃金數量 >= 3 : 開始製作金鏟子    # 素材用 svg 生成，需要製作動畫
        if 背包內水晶數量 >= 3 : 開始製作水晶鏟子    # 素材用 svg 生成，需要製作動畫

開始製作金鏟子、晶鏟子時，製作動畫不夠酷炫，同時希望有進度條
礦工背包裡的圖片都要注意有去背
第一個 row 設為禁止區域，禁止區域表示不能放任何物件，for loop 中表示為 continue
當黃金鏟子做出來時，施展十字攻擊，作用範圍包含整個 row 和 column，被波及的單位都會消失
當水晶鏟子做出來時，施展冰凍攻擊，作用範圍全地圖，施放動畫結束後，遊戲結束 ! 動畫用冰錐方式從地底往上出現，素材用 {專案中路徑}。仔細思考如何做出很有 UIUX 的動畫風格 !
